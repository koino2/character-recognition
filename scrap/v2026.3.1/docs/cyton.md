# Introduction
Cyton AI is a modern deep learning framework built specifically for Java developers.
It enables the creation, training, and deployment of neural networks directly within the Java ecosystem.

>Cyton AI aims to provide a lightweight, flexible, and developer-friendly approach to building machine learning systems in pure Java.

Cyton AI was created with a simple goal: bring powerful deep learning capabilities to the Java ecosystem.  
While most modern AI frameworks are built around Python, millions of production systems run on Java.
Cyton bridges that gap by enabling Java developers to design, train, and deploy neural networks
without leaving their comfort zone.

# Installation

Cyton AI is dependency-free and does not require a package manager.
It can be integrated directly into any Java project.
