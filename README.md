# Handwritten Character Recognition
#### This is a Flavortown project that shows how MLPS identify handwritten digits

## Features
- Uses my Cyton AI library for the MLP
- Has a GUI to draw images

## Usage
- Clone the repo
- Run Train to create the training file
- Run Main. (Make sure both have the same working directory)
- To add your own training samples, draw them in the same style in a 32x32 png and save it to the corresponding folder inside src/training
